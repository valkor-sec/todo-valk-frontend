📌 À propos du projet

Todo-Valk est une application de gestion de tâches complète avec authentification utilisateur.

Elle préssente les caractéristiques suivantes:

✔ Inscription / connexion sécurisée

✔ Création de tâches

✔ Marquer comme terminée

✔ Suppression de tâches

✔ Backend Node.js + Express

✔ Base de données MongoDB

✔ Authentification JWT

⚙️ Technologies utilisées

• CSS

• Node.js

• Express
